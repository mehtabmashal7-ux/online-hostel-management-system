# 🏨 Online Hostel Management System

A web-based Online Hostel Management System developed using PHP, MySQL, HTML, CSS, and JavaScript. This project helps manage hostel operations efficiently by providing separate dashboards for administrators and students.

## 📌 Features

- 🔐 Secure Login System
- 👨‍💼 Admin Dashboard
- 👨‍🎓 Student Dashboard
- 🛏️ Room Management
- 👥 Student Management
- 💳 Fee Management
- 📝 Complaint Management
- 📢 Notice Board
- 📊 Reports and Records

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- XAMPP

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mehtabmashal7-ux/online-hostel-management-system.git
   ```

2. Move the project folder to the `htdocs` directory of XAMPP.

3. Start **Apache** and **MySQL** from the XAMPP Control Panel.

4. Import the database into phpMyAdmin.

5. Open your browser and visit:

   ```
   http://localhost/onlinehostel/
   ```

## 🎯 Purpose

The main purpose of this project is to simplify hostel management by allowing administrators to manage students, rooms, fees, complaints, and notices while providing students with easy access to hostel-related services.
